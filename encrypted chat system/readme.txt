# Chat Application with Socket.io

The Chat Application with Socket.io is a real-time chat application built using Node.js, Express, and Socket.io. It allows users to engage in real-time messaging, group chat, private messaging, and receive message notifications.

## Features

- Real-time Messaging: Users can exchange messages in real-time with other connected users.
- Group Chat: Users can join and participate in group chat rooms.
- Private Messaging: Users can send private messages to specific users.
- Message Notifications: Users receive real-time notifications when new messages are received.
- Simple and Responsive UI: The user interface is designed to be user-friendly and responsive across different devices.

## Usage

1. Open the application in your web browser.

2. Enter your username and join the chat.

3. Start sending messages in the chat input box.

4. To send a private message, type `/msg <username> <message>`.

5. Enjoy real-time chat with other connected users!

## Technologies Used

- Node.js: A JavaScript runtime environment used for server-side development.
- Express: A web framework for Node.js used to build the server.
- Socket.io: A library that enables real-time, bidirectional communication between the server and clients.
- HTML/CSS: Markup languages used for structuring and styling the user interface.
- JavaScript: The programming language used for client-side functionality.
